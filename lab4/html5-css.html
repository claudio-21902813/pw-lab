CSS

Cascading Style Sheets (CSS) é um mecanismo para adicionar estilo (cores, fontes, espaçamento, etc.) a um documento web

tipos de seletores CSS
Elemento foi usado? Explicação
Universal done aplica o estilo a todos os elementos da pagina
Tipo done aplica o estilo do nome do marcador a utilizar
Classe report_problem aplica o estilo do nome de uma classe a utilizar
Identificador done aplica o estilo consoante o Identificador(muito util em Javascript)
Atributos done aplica o estilo consoante o atributo de um Elemento
Pseudo-classe report_problem aplica o estilo consoante o estado de um elemento
Pseudo-elemento report_problem aplica o estilo consoante em partes de elementos
combinadores de seletores CSS
Elemento foi usado? Explicação
Seletores descendentes ex: el el2 done – todos os "el2" que estão "dentro" de um el e foram usados nas paginas principais
Seletores filhos ex: el > el2 done – todos os "el2" que estão "logo dentro" de um el e foram usados nas paginas principais
Seletores adjacentes ex: el ~ el2 done todos os "el2" que estão colocados depois de um "el" e foram usados nas paginas principais
Seletores imediatamente adjacentes ex: el + el2 done todos os "el2" que estão colocados imediatamente depois de um "el" e foram usados nas paginas principais
Agrupação ex: el,el2 done os el e el2 ficaram com o mesmo estilo e foram usados nas paginas principais
